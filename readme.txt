【概要】
作りかけですがggspcをver0.0.1として公開します。
一言でいうとタイトル特化型のPS2エミュレータです。
汎用的なエミュレータと比べかなり高速に動作します。

もともとPS2エミュレーションの実験から始まり、
ggxx/のネット対戦を目標に2010年ごろまで作業してきましたが
色々思うところがあり、開発停止していたものを今回公開することにしました。

【ライセンス】
公開ライセンスはGPL2.0とします。
